# 🍃 Konoha Mission System - Desafio SOLID

Este projeto foi concebido para fins de estudo dos princípios **SOLID**, baseando-se na didática do livro *"SOLID para Ninjas"* de Mauricio Aniche. O sistema simula o gerenciamento de missões na Vila da Folha (Naruto), evoluindo de um código procedural para uma arquitetura robusta.

## 📌 O Projeto
O objetivo é processar a conclusão de missões ninjas, calcular recompensas (Ryōs), aplicar bônus de categoria e descontar impostos da vila.
